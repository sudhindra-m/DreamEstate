# DreamEstate
DreamEstate is a fully structured real estate website developed using pure HTML, aimed at providing a smooth and intuitive browsing experience for users exploring property-related services. The project showcases a clean design and well-organized navigation flow, making it a professional representation of how an online real estate platform functions.

The homepage acts as the central hub, guiding users to the main sections of the website through clearly labeled menus and interactive openable windows. Each section is designed to deliver relevant information effectively while maintaining simplicity in structure.

The key features of DreamEstate include:

1. Buy Section

This page displays properties available for purchase, organized neatly to help potential buyers explore various real estate options. The layout is structured to highlight the type of property, location details, and user-friendly descriptions that simulate how real-world property listings appear.

2. Rent Section

The rent page focuses on showcasing properties available for rental purposes. It provides a clear overview of room types, rental categories, or potential price ranges. This section is crafted to assist users who are looking for temporary or long-term rental spaces, maintaining an easy-to-navigate format.

3. Sell Section

This section is dedicated to property owners who wish to sell their real estate through the DreamEstate platform. It gives an overview of the selling process, the advantages of listing with DreamEstate, and the steps involved. Even though the project is static, this page imitates how sellers would interact with an online property listing service.
